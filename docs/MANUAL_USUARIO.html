1. Introducción

Cuadre de Archivos es una herramienta que permite comparar datos entre archivos Excel con múltiples hojas. El objetivo principal es validar que la información haya viajado correctamente de un archivo a otro.

¿Qué puede hacer?
  • Comparar estructuras vs datos
  • Validar campos específicos o todos
  • Generar reportes con columnas de resultado
  • Identificar registros faltantes

2. Requisitos del Sistema

ComponenteRequisito
JavaJDK 1.8 o superior
Sistema OperativoWindows 10/11
Memoria RAMMínimo 4GB

3. Instalación

  1. Descargue el archivo CuadreArchivos.zip
  2. Extraiga en cualquier ubicación de su computadora
  3. Asegúrese de tener Java instalado
  4. Ejecute haciendo doble clic en CuadreArchivos.bat

4. Cómo Usar

4.1 Seleccionar Archivos

  1. Haga clic en "Seleccionar Archivo" en el panel ARCHIVO 1
  2. Navegue hasta su archivo Excel y selecciónelo
  3. Repita el proceso para el panel ARCHIVO 2

4.2 Elegir Hojas

Después de seleccionar un archivo, el sistema muestra las hojas disponibles. Seleccione:

  • ARCHIVO 1: La hoja que contiene los datos originales
  • ARCHIVO 2: La hoja contra la cual comparar

4.3 Definir Columnas Clave

La columna clave es el campo que identifica cada registro de forma única:

Ejemplo:
Columna clave Archivo 1: NUMERO_REFERENCIA
Columna clave Archivo 2: REFERENCIA

4.4 Seleccionar Columnas a Comparar

En el panel derecho, seleccione las columnas que desea validar:

  • Use "Seleccionar Todo" para comparar todas las columnas
  • O seleccione solo las específicas que necesita validar

4.5 Ejecutar el Cuadre

  1. Haga clic en "EJECUTAR CUADRE" (botón rojo)
  2. Elija la ubicación para guardar el resultado
  3. Espere a que termine el procesamiento

5. Interpretar los Resultados

El archivo de salida incluye columnas adicionales al final:

ColumnaDescripciónValores posibles
CRUCE_ESTADOIndica si se encontró el registroENCONTRADO / NO ENCONTRADO
CRUCE_CLAVEValor de la clave utilizadaEl valor de la columna clave
CRUCE_VALIDACIONESTotal de campos comparadosNúmero
CRUCE_CORRECTOSCampos que coincidenNúmero
CRUCE_PORCENTAJE% de coincidencia0% - 100%

6. Ejemplos de Uso

Ejemplo 1: Validar Transacciones

Situación: Verificar que las transacciones de CREDIBANCO estén en el archivo del banco.

ARCHIVO 1: Credibanco_P2M_Breb.xlsx
  Hoja: Transacciones

ARCHIVO 2: Cuadre_07_04_2026.xlsx
  Hoja: Divide

Columna clave: NUMERO_REFERENCIA
Columnas a comparar: VALOR, FECHA, ESTADO

Ejemplo 2: Comparar Estructuras

Situación: Verificar que todos los clientes de la estructura estén en el archivo de datos.

ARCHIVO 1: Estructura_Clientes.xlsx
  Hoja: pgacl001 (estructura)

ARCHIVO 2: Datos_Clientes.xlsx
  Hoja: Clientes

Columna clave: ID_CLIENTE
Columnas a comparar: TODAS

7. Solución de Problemas

ProblemaSolución
La aplicación no abreVerifique que Java esté instalado: java -version
Error al abrir archivoAsegúrese de que el archivo no esté abierto en Excel
Mucho tiempo de procesamientoNormal para archivos grandes (>50,000 filas)
Columnas no aparecenVerifique que los encabezados estén en la primera fila
